# How To Start
Brief description or introduction of the project.

## Setup Environment
COPY env.example to .env and set up your .env file:

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_DATABASE=your_db
PORT=3000

```

## Installation
1. Install Dependencies
```
npm install
```

## Setup

1. Import Db
```
 npm run import
```


## RUN 

1.Project
```
npm run dev
```
